# Rive Editor Keyboard Shortcuts

## 1. Tool Selection

  Action                  Shortcut
  ----------------------- ----------
  Select Tool             V
  Translate (Move) Tool   Q
  Rotate Tool             W
  Scale Tool              E
  Rectangle Tool          R
  Ellipse Tool            O
  Pen Tool                P
  Artboard Tool           A
  Bone Tool               B
  Group Tool              G
  Freeze Tool             Y

------------------------------------------------------------------------

## 2. Object Editing & Selection

  Action                 Shortcut
  ---------------------- ----------------------
  Deep select            Cmd/Ctrl + Click
  Select behind object   Option/Alt
  Nudge object           Arrow Keys
  Edit vertices          Enter
  Duplicate selection    Cmd/Ctrl + D
  Group selection        Cmd/Ctrl + G
  Ungroup                Cmd/Ctrl + Shift + G
  Convert to Solo        Cmd/Ctrl + L
  Add to Solo            Cmd/Ctrl + L
  Rename                 Cmd/Ctrl + R

------------------------------------------------------------------------

## 3. Layer / Hierarchy Controls

  Action                       Shortcut
  ---------------------------- ----------------------------
  Move layer forward           Cmd/Ctrl + \]
  Move layer backward          Cmd/Ctrl + \[
  Move layer to top            Cmd/Ctrl + Alt/Option + \]
  Move layer to bottom         Cmd/Ctrl + Alt/Option + \[
  Select parent in hierarchy   Esc
  Select first child           Enter

------------------------------------------------------------------------

## 4. Timeline & Animation

  Action                      Shortcut
  --------------------------- ----------------------
  Move playhead               ,
  Move selected keys          Alt/Option + ,
  Skip to keys                Cmd/Ctrl + ,
  Jump playhead to start      Cmd/Ctrl + Shift + ,
  Jump playhead to end        Cmd/Ctrl + Shift + .
  Play state machine          Shift + Space
  Reveal keys for selection   U

------------------------------------------------------------------------

## 5. View & Navigation

  Action                    Shortcut
  ------------------------- ---------------
  Fit selection to screen   F
  Zoom tool                 Z
  Zoom in                   \+
  Zoom out                  \-
  Pan canvas                Space + Drag
  Marquee zoom              Hold Z + drag

------------------------------------------------------------------------

## 7. Vertex / Path Editing

  Action                            Shortcut
  --------------------------------- ---------------------------

  Detach handle                     Alt/Option + Click handle
  Delete vertex                     Alt/Option + Click vertex

## 9. Code Editor Shortcuts (Rive scripting panel)

  Action                   Shortcut
  ------------------------ --------------------------
  Find                     Cmd/Ctrl + F
  Find next                Cmd/Ctrl + G
  Find previous            Cmd/Ctrl + Shift + G
  Find in files            Cmd/Ctrl + Shift + F
  Toggle comment           Cmd/Ctrl + /
  Insert line below        Cmd/Ctrl + Enter
  Insert line above        Cmd/Ctrl + Shift + Enter
  Move line up/down        Alt + ↑ / ↓
  Duplicate line           Shift + Alt + ↑ / ↓
  Multi‑cursor             Alt + Click
