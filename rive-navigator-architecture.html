An AI co-pilot for Rive with two core loops: a vision-based UI agent that understands and operates the editor in real time, and a Nano Banana SVG Generator that turns prompts into concept art, traces it into sanitized SVG, pastes it into Rive, and hands off to the animation agent. 01 — Context
Problem & Solution

The Problem

Rive is a powerful animation editor with a steep learning curve. Designers and developers struggle to navigate multi-step workflows—state machines, keyframes, transitions, imports, inspector edits—without extensive documentation reading. The editor is a Flutter Web canvas app, making standard browser automation unreliable, and pure in-editor generation quickly hits a ceiling once the task requires assets more complex than simple shapes.

The Solution

Rive Navigator combines a multimodal UI agent with a dedicated prompt-to-SVG asset pipeline. It sees the editor via screenshots, understands Rive's UI layout spatially, answers questions contextually, guides users step-by-step with visual cursors, autonomously executes trusted browser events, and can generate a concept image, trace and sanitize SVG, paste it into Rive, then continue with animation.

Target Users

  • Designers learning Rive for interactive UI and motion design
  • Developers integrating Rive animations into production apps
  • Teams building complex state machines and transition logic

Key Differentiators

  • Multimodal vision—Gemini analyzes live editor screenshots and grounds decisions in current UI state
  • Agentic control—AI executes clicks, keystrokes, typing, hover, drags, and paste commands directly via CDP
  • Prompt-to-SVG creation—Nano Banana SVG Generator produces concept art, vectorizes it, sanitizes it, and imports it into Rive
  • Runtime context engine—small stable policy + per-turn procedure cards + output contracts tuned for Flash reliability
  • Documentation grounding—285 .mdx files with section-level scoring, extracted steps, visual-dependency hints, and image refs
  • Push-to-talk + TTS—hold Ctrl+Space, speak, release to send. Full text-free loop with subtitle overlay

Core Runtime Loops

Rive Navigator now operates as two coordinated runtime loops. The first is the vision-guided agent loop that understands and controls the editor. The second is the Nano Banana SVG loop that turns a prompt into a Rive-imported asset, then hands the new editor state back to the same agent for animation.

Nano Banana SVG Loop

This pipeline is intentionally separate from `/api/chat`. 06 — Actions
CDP Action Types

All actions dispatched via Chrome Debugger API producing trusted browser events. Critical for Flutter Web which ignores synthetic JavaScript events.

click / doubleclick

mousePressed + mouseReleased at (x%, y%). Doubleclick uses 3-phase: click to select, 400ms pause, then double-click for rename/edit mode.

key

keyDown + keyUp with modifier support (meta, ctrl, shift, alt). Preferred over clicking toolbar icons which are small and tightly spaced.

type

Atomic field entry: triple-click to focus + select all, verify text field focused, Backspace, Input.insertText, Enter. Safe for canvas apps.

drag

mousePressed at start, 10 animated mouseMoved steps at 30ms intervals, mouseReleased at end. Used for drawing shapes and creating transitions.

hover

mouseMoved without clicking + 500ms wait. Triggers hidden UI elements like state machine connector dots that only appear on hover.

wait

Configurable delay for UI transitions to complete. No browser interaction dispatched. Used between complex action sequences.

08 — Capabilities
Features
Functional

Vision-Based Rive Navigation

Gemini multimodal vision identifies panels, selected objects, active tools, animation state, and spatial positions in the Rive editor, then emits validated CURSOR or ACTION contracts.

Step-by-Step Task Guidance

Structured task plans with progress tracking, step verification, back/skip/end controls, and explicit core-editor guidance so Flash has a strong baseline even before targeted retrieval kicks in.

Nano Banana SVG Generator

A dedicated asset workflow generates a concept preview, lets the user approve it, traces the image with vtracer, sanitizes the SVG, and pastes it directly into Rive for immediate animation.

Documentation Grounding

Section-level index of 285 Rive .mdx files with multi-factor scoring, step extraction, visual-dependency detection, and image reference attachment. Prevents hallucination by grounding answers in official docs.

Non-Functional

Audio Narration

Gemini TTS with sentence pipelining for low-latency playback. Pre-fetches next sentence while current plays. Subtitle overlay on canvas.

Push-to-Talk Voice

Hold Ctrl+Space for hands-free input — pulsing mic overlay with live transcript on the Rive canvas. Web Speech API runs in the content script (sidepanels lack mic access), with results relayed via chrome.runtime messaging. Enables a fully text-free voice-in → voice-out workflow.

Safety & Robustness

Action limit (150), stuck detection (2 repeats), invalid-action recovery, coordinate validation (% vs px), text field focus verification before destructive keystrokes, and agent-side restrictions for high-risk rename flows.

Import Reliability

SVG import uses rich clipboard MIME data with text fallback, explicit CDP Paste commands, tab/window focus restoration, and post-import screenshot capture so the agent continues from the real editor state.

Debug Observability

Detailed request/response logging, event-by-event agent execution tracing, asset preview timeout handling, upstream overload surfacing, and optional screenshot persistence for debugging.

09 — Deploy
Deployment

Local Development

1
Python venv — python3 -m venv venv && source venv/bin/activate && pip install -r requirements.txt && pip install vtracer
2
API key — Set GOOGLE_API_KEY in agent/.env
3
Start backend — python -m uvicorn agent.server:app --reload --port 8000
4
Load extension — chrome://extensions → Developer Mode → Load unpacked → chrome-extension/
5
Open Rive — Navigate to rive.app, click extension icon to open sidebar, then test both chat and the Nano Banana SVG Generator

Production

1
Docker image — Build the FastAPI backend for Cloud Run and include the vectorization dependency so /api/assets/vectorize works in production too.
2
Deploy — gcloud run deploy rive-navigator --source . --region us-central1
3
Env vars — Set GOOGLE_API_KEY plus runtime knobs like ASSET_PREVIEW_TIMEOUT_SECONDS and doc-image attachment flags as needed.
4
Update extension — Point the extension API URL at Cloud Run and package the extension as an unpacked test build or optional unlisted Chrome Web Store listing for judges.
5
Judge-ready demo — Show both flows: vision-guided editor control and prompt-to-SVG import followed by animation inside the same live session.

Prerequisites

  • Python 3.11+
  • Google Chrome 114+ (sidePanel API)
  • Gemini API key (free tier available)
  • vtracer available in the backend environment for SVG generation
  • Google Cloud account (for production deployment)
